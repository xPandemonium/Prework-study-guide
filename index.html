Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Within the body elements, you can seperate the sections of the page with the <\section\> element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can use the special character * to signify that you want to format all elements on the page.
  • You can define an element on the page with a class in HTML that can later be used in CSS to format all the elements with that same class.

Git

  • 'git status' checks what branch we are currently on as well showing any pending stange/commits changes.
  • 'git checkout -b branch-name' creates a new branch and switches to it
  • 'git add -A' stages all changes done
  • 'git commit -m "commit coment" ' commits all the stanged changed to the current branch we are one
